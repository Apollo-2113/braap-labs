Architecture / Signal Pipeline
RAW SENSOR  →  EUCLIDEAN MAGNITUDE √(ax²+ay²+az²)  →  LPF (EMA α)  →  STATE MACHINE  →  JUMP EVENT
Gyroscope stream is processed in parallel → magnitude √(gx²+gy²+gz²) → peak tracked per-jump.
Orientation-agnostic: works regardless of phone placement in pocket (portrait/landscape/inverted).
On mobile: DeviceMotionEvent @ ~60Hz. Production iOS: CMMotionManager @ 100Hz. Android: TYPE_LINEAR_ACCELERATION + TYPE_GYROSCOPE.
